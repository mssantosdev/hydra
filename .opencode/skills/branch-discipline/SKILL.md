name: branch-discipline
description: Keep implementation work isolated to the assigned branch and task boundary.
---
## Rules

- verify the assigned branch and task before editing
- avoid unrelated edits in shared hotspot files
- prefer helper extraction or localized changes
- if another branch's area must be touched, call that out explicitly before proceeding

## Use when

Use this for implementation tasks that run in parallel across multiple worktrees or branches.
