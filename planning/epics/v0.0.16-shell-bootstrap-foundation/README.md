# Epic v0.0.16 Shell Bootstrap Foundation

- Status: `active`
- Target release: `v0.0.16`

## Objective

Establish the first durable multi-agent development wave for Hydra, focusing on shell/completion architecture, CLI version visibility, and `hydra new` hardening.

## Included

- file-based shell integration and completion flow
- CLI version visibility
- `hydra new` hardening and polish
- planning/agent framework required to execute this epic with Hydra itself

## Excluded

- local-only file tracking/copying
- broader local repo import lifecycle
- broad documentation cleanup beyond what the included features require
